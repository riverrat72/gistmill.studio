What We Do

The Gist Mill is a small, independent software studio founded and operated by a physical therapist. Our work centers on building lightweight, clinician-friendly documentation tools and exploring new ways to improve communication in clinical environments using low-compute, accessible technology.

Our current focus is the development of a digital clipboard system that uses stylus-based handwriting recognition to streamline point-of-care documentation. This tool is being designed as an adjunct to existing EMR workflows, with no formal agreements or integrations in place at this time.

Active and exploratory projects include: a speech-to-text engine tailored for noisy clinic environments designed as an optional, push-to-activate speech-to-text tool, used only when the therapist intentionally holds a physical button to dictate longer notes or descriptions that would be impractical to write by hand. an early research phase model of a dual-vision “astigmatism” model intended to improve generalization on messy, real-world handwriting.
